Introduccion

La recopilación y codificación de datos es una de las fases más importantes en cualquier proyecto de investigación, ya que la calidad y precisión de los datos obtenidos pueden afectar significativamente los resultados finales y la validez de las conclusiones. En este blog, exploraremos los elementos esenciales a considerar para una recopilación de datos exitosa y una codificación de datos precisa.

Instrumentos

Los 3 más usados en el campo de la recolección de datos

Cuestionario

El cuestionario es una herramienta diseñada para la recolección de datos cuantitativos , y se utiliza mucho en la investigación, ya que es un buen instrumento de investigación para recolectar datos estandarizados y hacer generalizaciones.

Entrevistas

Las entrevistas son una herramienta principalmente para la recolección de datos cualitativos y son populares como instrumentos para recopilar información debido a su flexibilidad. Las interacciones que se generan en una entrevista pueden presentarse de forma estructurada o semiestructurada para generar ideas y conceptos.

Observación

La observación es otro instrumento para la recolección de datos cuantitativos o cualitativos basado en la observación de los individuos en un entorno o situación natural. En este método los datos se pueden registrar mediante grabaciones de video o audio y notas de campo. La participación del observador puede ser limitada o implicita.

Herramientas

En la evaluación de Proyectos

Es importante seleccionar las herramientas adecuadas en función de los objetivos y características del proyecto, y utilizarlas de manera integrada para obtener una evaluación completa y precisa del proyecto.

Algunas de esas herramientas son:

  1. Análisis costo-beneficio (ACB)
  2. Análisis costo-efectividad (ACE)
  3. Análisis de sensibilidad
  4. Análisis de Riesgos
  5. Análisis de Ciclo de Vida(ACV)

Criterios de Categorización

Según los párametros objetivos

La categorización de la información es el proceso de agrupar la información recogida en base a ciertos criterios, con el fin de analizarla y obtener conclusiones. Para llevar a cabo la categorización de la información, se deben establecer criterios de categorización que permitan clasificar la información de manera objetiva. Estos criterios pueden ser establecidos de antemano o surgir a medida que se va recopilando la información

Elementos Éticos

En el desarrollo de proyectos

En el desarrollo de cada proyecto de investigación, es esencial tener en cuenta los principios éticos, tanto en el diseño como en la ejecución del proyecto. Entre los aspectos éticos que deben ser considerados se encuentran:

Consentimiento informado: se debe obtener el consentimiento informado de todos los participantes antes de que se les solicite su participación en el proyecto. El consentimiento debe ser libre, voluntario, informado y dado por escrito.

Confidencialidad y privacidad: se debe proteger la privacidad y confidencialidad de los participantes, asegurando que la información que se recoja sea manejada con discreción y no se revele a terceros.

Beneficencia: se debe valorar en todo momento el posible bienestar de los participantes, minimizando cualquier riesgo o daño potencial.

Honestidad y rigor científico: se debe abordar el proyecto de forma cuidadosa, respetando los conocimientos previos y principios científicos, evitando cualquier tipo de manipulación o falsificación de datos.

Sistemas de Codificación y Procesamiento de Datos

Definiciones, Descripciones, Alcances y Limitaciones

Los sistemas de codificación y procesamiento de datos son herramientas importantes en la investigación. La codificación es un procedimiento técnico mediante el cual los datos obtenidos se clasifican en categorías y se traducen en símbolos, ya sean cifras o letras, para poder tabularlos rápidamente.

Sistema de codificación binario:

Definiciones:

El sistema de codificación binario es un sistema numérico que utiliza únicamente dos símbolos, 0 y 1, para representar información.

Descripción:

En este sistema, cada dígito se llama "bit" y se utiliza para representar un valor binario, que puede ser 0 o 1. Los bits se agrupan en conjuntos de 8 para formar un byte, que es la unidad de almacenamiento básica en la mayoría de los sistemas informáticos.

Alcance:

El sistema de codificación binario se utiliza en la mayoría de los sistemas informáticos, como computadoras y dispositivos móviles, para procesar y almacenar información.

Limitaciones:La representación binaria de información puede resultar difícil de entender para personas que no están familiarizadas con ella. Además, algunos tipos de información, como imágenes y sonidos, requieren grandes cantidades de bits para ser representados, lo que puede requerir mucho espacio de almacenamiento.

Sistema de codificación UNICODE:

Definiciones:

El sistema de codificación Unicode es un sistema de codificación de caracteres que utiliza más de 100.000 caracteres para representar todos los idiomas del mundo.

Descripción:

El sistema Unicode se utiliza en muchos sistemas informáticos para representar texto en diferentes idiomas y alfabetos. Cada carácter tiene un número único asociado que se utiliza para representarlo en el sistema Unicode.

Alcance:

El sistema de codificación Unicode permite la representación de texto en muchos idiomas y alfabetos diferentes. Es compatible con muchos otros sistemas de codificación y se puede utilizar para intercambiar información entre diferentes sistemas informáticos.

Limitaciones:

El sistema de codificación Unicode requiere más espacio de almacenamiento que otros sistemas de codificación debido a la gran cantidad de caracteres que utiliza. Además, algunos programas y sistemas informáticos pueden no ser compatibles con todos los caracteres Unicode.

Sistema de codificación ASCII:

Definiciones:

El sistema de codificación ASCII (American Standard Code for Information Interchange) es un sistema de codificación de caracteres que utiliza 7 bits para representar los caracteres más comunes del alfabeto inglés, como letras, números y símbolos

Descripción:

El código ASCII se utiliza para representar texto en muchos sistemas informáticos, como sistemas operativos y aplicaciones. Cada carácter tiene un valor numérico asociado que se utiliza para representarlo en el sistema ASCII.

Alcance:

El sistema de codificación ASCII es ampliamente utilizado en sistemas informáticos para representar texto en inglés y otros idiomas que utilizan caracteres del alfabeto latino. Es compatible con muchos otros sistemas de codificación y se puede utilizar para intercambiar información entre diferentes sistemas informáticos.

Limitaciones:

El sistema de codificación ASCII tiene una capacidad limitada para representar caracteres en idiomas que utilizan alfabetos diferentes al latino. Además, no puede representar caracteres especiales que se utilizan en algunos idiomas.
